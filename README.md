# Projeto_agrinho
Imagine que a Terra é um grande jogom garantindo que as futuras gerações também possam desfrutar deles.Nesse grande jogo da vida, não existem vencedores individuais. A vitória depende da colaboração de todos. Pequenas atitudes, como usar menos plástico, plantar árvores e separar materiais recicláveis, podem fazer uma enorme diferença
